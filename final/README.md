# 编译器构造实验

## 提交内容

1. 设计报告

   > report.pdf

2. 源代码压缩包

   > src文件夹

3. 可执行文件

   > main.exe

4. 源码测试文件（待编译的程序，程序必须覆盖所支持的文法单元）

   > demo.pas

5. 测试输出文件（目标代码文件）

   > 终端输出

6. readme.md（程序复现说明书)

   > 本文件

## 程序复现说明

makefile包含程序的制作规则。

编译

```shell
mingw32-make
```

运行

```shell
mingw32-make test
```
